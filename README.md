# Computational Reproduction: Direct Preference Optimization (DPO)

**Lead Researcher:** Giovanna Caroline Monteiro
**Concentration Domain:** Computational Neuroscience / Artificial Intelligence
**Academic Level:** Ph.D. / Advanced Research

## Overview
This independent research package serves as a comprehensive template for understanding and computationally reproducing a fundamental scientific discovery in neural model alignment. The objective is to simulate the complete scientific workflow, from reviewing prior literature to reproducing the results of the target paper.

## Target Paper
*Direct Preference Optimization: Your Language Model is Secretly a Reward Model* (Rafailov et al., Stanford University, 2024).

## How to Use This Package
This repository is self-contained. To reproduce the reported results:
1. Ensure you have Python 3.9+ installed.
2. Install the required dependencies by running: `pip install -r requirements.txt`
3. Read `1_Literature_Review.md` to understand the research lineage.
4. Consult `2_Core_Extraction.md` for the methodologies and mathematical hypotheses.
5. Execute the experiment with the command: `python 3_experiment_reproduction.py`
